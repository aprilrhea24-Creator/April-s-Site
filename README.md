# Aura App Studio

Private developer portfolio and B2B SaaS storefront for April's custom operational app work.

## Core Routes

- `/` Agency landing page for custom automated business apps.
- `/solutions` Showcase grid for reusable app-system templates.
- `/consultation` Build-request funnel for prospective clients.
- `/workspace` Admin-only dashboard preview for April.
- `/client-dashboard` Authenticated client dashboard preview.
- `/about` Flexible professional agency boilerplate.
- `/policies` SaaS and freelance project policy language.

## Development

```bash
npm install
npm run build
npm run dev
```

The current foundation prioritizes a premium glassmorphism interface, conversion-focused service flow, guarded private
workspace access, and a clean public brand surface for April's Site.
